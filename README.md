# Alipay_demo
### 一、说明：
- 本Demo使用的是python-alipay-sdk，github连接：https://github.com/fzlee/alipay
- 本Demo下载即用，对以下支付功能兼容：（只需更改对应的方法即可）
    - 电脑网站支付: alipay.trade.page.pay
    - 手机网站支付: alipay.trade.wap.pay
    - App支付: alipay.trade.app.pay

### 二、开发环境：
- Django 2.0
- python 3.6
